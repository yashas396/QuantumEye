**Gate Explanation:**

| Gate | Symbol | Purpose |
|------|--------|---------|
| **RY(Î¸)** | Rotation around Y-axis | Encode input data as angles |
| **RX(Î¸)** | Rotation around X-axis | Learnable parameter |

**Code:**
```python
@qml.qnode(dev, interface="torch")
def qdt_circuit(inputs, params):
    # Angle Embedding: Encode 4 features as rotation angles
    for i in range(n_qubits):
        qml.RY(inputs[i], wires=i)
    
    # Variational Layers: 10 layers of rotations + entanglement
    for layer in range(n_layers):
        for i in range(n_qubits):
            qml.RX(params[layer, i, 0], wires=i)
            qml.RY(params[layer, i, 1], wires=i)
            qml.RZ(params[layer, i, 2], wires=i)
        
        # Ring entanglement: q0-q1-q2-q3-q0
        for i in range(n_qubits):
            qml.CNOT(wires=[i, (i + 1) % n_qubits])
    
    return qml.probs(wires=range(n_qubits))  # 16 probabilities
```
